## **3. Step-by-Step Guide with 10 Prompts**
Break down the project into clear phases with corresponding prompts.

### **Prompt 1: Understanding the Data**
*"I have an Excel file with energy consumption data (columns: Date, Consumption, Peak Time, etc.). Can you help me understand how to parse it and what data structure would best represent this in JSON?"*

### **Prompt 2: Parsing the Excel File**
*"Generate a Python script using pandas to parse an Excel file and convert it into a JSON file. Ensure the JSON format retains column names and supports nested structures if needed."*

### **Prompt 3: Writing a Bash Script**
*"Write a bash script to automate the process of running the Python script and saving the JSON output to a specific directory."*

### **Prompt 4: Setting Up Angular 19**
*"What are the steps to set up a new Angular 19 application? Please include commands and explain how to install dependencies."*

### **Prompt 5: Installing Angular Material**
*"How can I install and configure Angular Material in an Angular 19 project?"*

### **Prompt 6: Adding JSON Data**
*"How can I import a local JSON file containing energy consumption data into my Angular project and use it as a data source?"*

### **Prompt 7: Creating a Material Table**
*"Generate Angular component code to display energy consumption data in a Material table with sorting and pagination features."*

### **Prompt 8: Adding a Chart**
*"Using Chart.js, provide Angular component code to create a line chart that visualizes energy consumption trends over time."*

### **Prompt 9: Styling and Enhancements**
*"Suggest UI enhancements for the Angular app using Material components, such as a responsive layout or a toolbar with filters."*

### **Prompt 10: Optimizing for Deployment**
*"How can I build and deploy the Angular app for production, ensuring the best performance?"*
