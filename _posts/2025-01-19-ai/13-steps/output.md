*"I have energy consumption data in JSON format with the following structure:"*

```json
  { "date": "2024-01-01T00:00:00.000Z", "consumption": 0.065, "generation": 0.012 }
]
```
*"How should I structure this data in an Angular application? Which TypeScript interface best represents it?"*

*"Write an Angular 19 service (`EnergyReportService`) that loads the energy data from `assets/energy-report.json`. Use `HttpClient` and return an `Observable<EnergyReport[]>`."*

### **🔹 Prompt 4: Building a Material Table to Display Energy Data**
*"Generate an Angular Material table (`EnergyTableComponent`) that displays the JSON data with columns for Date, Consumption, and Generation. Include sorting and pagination."*
